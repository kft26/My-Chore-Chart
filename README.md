# Chore Quest 🏆

Single-file family chore & learning tracker. No build step needed.

## Deploy to GitHub Pages (3 steps)

### 1. Create GitHub repo
Go to https://github.com/new → name it `chore-quest` → Create

### 2. Push these files
```bash
cd chore-quest-html
git init
git add .
git commit -m "Chore Quest"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/chore-quest.git
git push -u origin main
```

### 3. Enable GitHub Pages
- Go to your repo → Settings → Pages
- Source: **Deploy from a branch**
- Branch: **main** / **/ (root)**
- Save

Your app is live at: `https://YOUR_USERNAME.github.io/chore-quest/`

## Install on iPad
1. Open Safari → go to your GitHub Pages URL
2. Tap Share button (square with arrow)
3. Tap "Add to Home Screen"
4. Done — launches full-screen like an app
